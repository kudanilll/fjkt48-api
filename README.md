# FJKT48 API Documentation

FJKT48 API Documentation

## Getting Started

First step, clone this repo

```bash
$ git clone -b development https://github.com/achmaddaniel24/fjkt48-api.git fjkt48_api
$ cd fjkt48_api
```

Install `rimraf` package

```bash
$ npm i -g rimraf
```

Install all packages

```bash
$ npm i
```

Setup prepare husky

```bash
$ npm run prepare
```

Run server

```bash
$ npm run dev
```
